Вертикальная группа радио инпутов с заголовком
```
<RadioGroup>
    {['Один', 'Два', 'Три', 'Четыре'].map(text =>
        <Radio
            text={ text }
            key={ text }
            value={ text }
        />
    )}
</RadioGroup>
```

Та же группа, теперь с заголовком и ошибкой
```
<RadioGroup error='Обязательно' label={ <Label size='m'>Сколько?</Label> }>
    {['Один', 'Два', 'Три', 'Четыре'].map(text =>
        <Radio
            text={ text }
            key={ text }
            value={ text }
        />
    )}
</RadioGroup>
```
